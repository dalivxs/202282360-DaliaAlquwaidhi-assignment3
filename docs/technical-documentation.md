# Technical Documentation – Assignment 1

## Overview
This project is a responsive personal portfolio website developed using HTML, CSS, and JavaScript. It serves as the foundation of a professional portfolio and demonstrates basic front-end development skills.

## Technologies Used
- HTML5 – Structure and semantic layout
- CSS3 – Styling, layout, and responsive design using Flexbox and media queries
- JavaScript – Interactivity (theme toggle, greeting message, form handling)

## Website Structure

### index.html
Contains the main structure of the website:
- Header with navigation
- Hero section
- About section
- Projects section (2 project cards)
- Contact form
- Footer

### css/styles.css
Handles:
- Layout and spacing
- Responsive behavior for mobile and tablet
- Dark/light theme styling
- Card and button design

### js/script.js
Implements:
- Dynamic greeting based on time of day
- Dark/light theme toggle with localStorage
- Basic contact form interaction (front-end only)

## Responsive Design
The layout adapts to:
- Desktop (multi-column layout)
- Tablet (adjusted spacing)
- Mobile (stacked layout)

Media queries ensure proper scaling and usability across devices.

## Limitations
- The contact form does not send data to a server (no backend required).
- Project links are placeholders.
- Images are placeholders.

## Future Improvements
- Connect contact form to a backend service.
- Deploy using GitHub Pages.
- Add animations and improved UI components.
