# Assignment 2 – Personal Portfolio (Enhancements)

## Project Description
This project is an improved version of my personal portfolio website from Assignment 1. In this version, I added more interactivity and better user experience using JavaScript, dynamic content, and smooth visual effects.

## Features
- About, Projects, and Contact sections
- Dark/light theme toggle with saved preference using localStorage
- Live project search feature
- Contact form validation with user feedback
- Responsive design for desktop, tablet, and mobile
- Hover effects and smooth transitions for a more modern experience

## Folder Structure
- `index.html` – main page
- `css/styles.css` – styling, layout, responsiveness, and animations
- `js/script.js` – theme toggle, search functionality, and form validation
- `assets/images/` – placeholder images
- `docs/ai-usage-report.md` – AI usage report
- `docs/technical-documentation.md` – technical details

## How to Run Locally
1. Clone the repository:
   - `git clone <YOUR_REPOSITORY_LINK>`
2. Open the folder in VS Code.
3. Open `index.html` in your browser.
